## 一、布局计算

### 居中公式

```
单元素居中：x = (容器宽 - 元素宽) / 2 + 容器左边距

多元素等宽：w = (W - (n+1) × gap) / n
           第 i 个 x = 容器左边 + gap + i × (w + gap)

指定宽度等距：gap = (W - n × w) / (n + 1)
```

### 间距标准（基于 8px 网格）

| 类型 | 间距 |
|------|-----|
| 最小间距 | 8px |
| 组件间距 | 16px |
| 层间距 | 20-24px |
| 边距 | 32px |

### 尺寸一致性

- 同排组件宽高必须一致
- 图标统一 45×45
- 卡片高度统一 80px

### 不完整行处理

最后一行居中：`x = 容器左边 + (容器宽 - m × w - (m-1) × gap) / 2`

## 二、连线质量

### 基本原则

- 方向统一：上→下 或 左→右
- 避免回头线和交叉
- 交叉过多 → 调整组件位置，不是调连线

### 通道预留

相邻组件间留出 30-50px 连线通道。

### 锚点选择

| 场景 | 锚点 |
|------|------|
| 上层到下层 | 上边出，下边进 |
| 左侧到右侧 | 左边出，右边进 |

### 线宽语义

| strokeWidth | 语义 |
|-------------|------|
| 2.5-3 | 主链路 |
| 2 | 普通数据流 |
| 1.5 | 次要连接 |

### 颜色编码

| 颜色 | Hex | 语义 |
|------|-----|------|
| 蓝 | `#2196F3` | 外部流量 |
| 橙 | `#FF9800` | 路由/均衡 |
| 红 | `#FF5722` | AI/ML 链路 |
| 绿 | `#43A047` | 业务逻辑 |
| 紫 | `#7B1FA2` | 数据层调用 |

## 三、成品感打磨

### 对齐

- 所有坐标为整数
- 同排元素 y 坐标相同
- 同列元素 x 坐标相同

### 字体层级

| 元素 | 字号 | 样式 |
|------|-----|------|
| 大标题 | 18-20px | 加粗 |
| 层标题 | 14-16px | 加粗 |
| 卡片标题 | 12-14px | 加粗 |
| 连线标签 | 9-10px | 常规 |

### 颜色克制

- 底色最多 4-5 种
- 连线颜色最多 3-4 种
- 按职责分类选色

### 细节装饰

```
圆角：rounded=1;arcSize=8（温和）/ arcSize=3（微圆角）/ rounded=0（直角）
阴影：shadow=1（轻阴影）/ shadow=0（无阴影，更现代）
边框：strokeWidth=2（中等）/ strokeWidth=1（细）
```

## 四、快速检查清单

**布局**：
- [ ] 同排宽高一致
- [ ] 间距系统化
- [ ] 调用方在上/左

**连线**：
- [ ] 方向统一
- [ ] 无明显交叉
- [ ] 颜色有语义

**视觉**：
- [ ] 颜色克制
- [ ] 字号层级分明
- [ ] 整体风格统一
