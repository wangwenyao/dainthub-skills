### 附录 B：技术术语表

**触发条件：**
扫描第 1、5 章和附录 A，如出现以下技术术语，在此集中解释：

| 常见术语 | 业务语言解释 |
|---------|------------|
| JWT Token | 用户登录后获得的访问凭证，{值}小时内无需重复登录，持续操作时自动续期 |
| RBAC | 按角色分配权限的方式，财务角色只能看到财务功能，业务角色只能操作自己部门的数据 |
| Redis | 缓存服务器，加速常用数据的读取，故障时系统自动切换为直接查询数据库 |
| Nginx | 流量分发服务器，将用户请求分配到多个应用实例，单个实例故障时自动切换 |
| MySQL | 关系型数据库，存储所有业务数据 |
| Quartz | 定时任务调度器，按设定时间自动执行任务（如每日凌晨生成报表）|

**生成规则：**
- 只解释在第 1、5 章、附录 A 中实际出现的术语
- 解释用业务语言，参考 writing.md 的转化示例
- 如第 3、4 章出现的术语未在业务章节出现，不需要在此解释
